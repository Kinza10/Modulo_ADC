Modulo Sampling ADC

High Dynamic Range ADC Architecture for Passive Radar

The Problem

Limited Dynamic Range

Traditional ADCs clip signals when input amplitude exceeds reference voltage. This causes information loss in wideband signal acquisition systems.

Passive Radar Challenge

Passive radar receives signals from broadcast transmitters which vary greatly in amplitude. Conventional ADCs require extremely high resolution and power to capture these signals accurately.

Our Solution

Our design uses a Modulo Sampling ADC architecture. A comparator detects when the input exceeds reference limits, then a feedback loop folds the signal back into the safe voltage range. The number of folds is tracked digitally and later used to reconstruct the original high-dynamic-range signal.
